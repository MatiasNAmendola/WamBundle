Warren's Abstract Machine - Implementation by Stefan Buettcher - Ported to PHP by Trismegiste

developed: December 2001 until February 2002 ported: July 2012

CompilerStructure contains the CompilerStructure class, which is needed for transforming the input vector (Prolog program) to the output Program (WAM code) via a certain program structure graph.

Each instance of CompilerStructure represents a node in the program graph of the original Prolog program. Every node is of a certain type (see constants below).
