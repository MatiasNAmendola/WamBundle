Warren's Abstract Machine - Implementation by Stefan Buettcher developed: December 2001 until February 2002

Translated from Java to PHP 5.3 by Florent Genette (Trismegiste) http://github.com/Trismegiste/WamBundle June to July 2012 (yes, 10 years after the original version !)

WAM.php contains an abstract WAM without input or output WAM code operations are described in Ait Kaci: Warren's Abstract Machine -- A Tutorial Reconstruction
