Warren's Abstract Machine - Implementation by Stefan Buettcher - Ported to PHP by Trismegiste

developed: December 2001 until February 2002 ported: July 2012

Statement contains the class Statement, representing a single line of WAM code, e.g. "true: proceed".

Statement class implements WAM code statements a statement looks like this: [label:] operator operand1 [operand2 [operand3]] label, op2 and op3 may be omitted. label is needed for jumps (calls)
