Warren's Abstract Machine - Implementation by Stefan Buettcher - Ported to PHP by Trismegiste

developed: December 2001 until February 2002 ported: July 2012

QueryCompiler.java contains the QueryCompiler class, a child-class of the Compiler class. QueryCompiler compiles user-written queries into WAM code for execution.
